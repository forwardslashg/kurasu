# Kurasu

#1 Anime streaming utility

## Prerequisites

- [Node.js](https://nodejs.org/)
- npm
- [mpv](https://mpv.io/)


## Installation

```bash
git clone https://github.com/forwardslashg/kurasu.git
cd kurasu
npm i
node .
```

## Configuration

You can optionally add extra mpv arguments in `config.json`:

```json
{
  "mpvExtraArgs": "--67"
}
```

Leave it empty or remove it to run mpv without extra arguments.
